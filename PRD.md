# 결혼매력평가 시스템 (MAAS - Marriage Attractiveness Assessment System)
## Product Requirements Document v1.0

---

## 1. 제품 개요

### 1.1 비전
재미있고 가벼운 심리테스트 형식의 "결혼매력평가" 서비스를 통해 사용자들이 자신의 매력을 평가받고, 결과를 SNS에 공유하며 자연스럽게 바이럴을 일으키는 웹 기반 엔터테인먼트 플랫폼

### 1.2 목표
- **사용자 참여**: 5분 이내 완료 가능한 간단하고 재미있는 테스트
- **바이럴 효과**: SNS 공유를 통한 자연스러운 확산
- **높은 재방문율**: 친구들과 비교하고 싶은 욕구 자극
- **수익화 가능성**: 향후 프리미엄 기능 추가 가능

### 1.3 타겟 사용자
- **주 타겟**: 20-35세 미혼 남녀
- **부 타겟**: 결혼에 관심 있는 모든 연령층
- **페르소나**: SNS 활동이 활발하고, 심리테스트를 즐기며, 친구들과 결과를 공유하는 것을 좋아하는 사용자

## 2. 핵심 기능

### 2.1 평가 시스템

#### 2.1.1 질문 구성 (15-20문항)
**카테고리별 평가 항목:**

1. **외적 매력** (3-4문항)
   - 자기관리 정도
   - 패션 감각
   - 건강/운동 습관
   - 첫인상

2. **경제력** (3-4문항)
   - 직업 안정성
   - 재산 관리 능력
   - 미래 계획
   - 소비 습관

3. **성격/인성** (4-5문항)
   - 배려심
   - 유머 감각
   - 감정 조절 능력
   - 공감 능력
   - 책임감

4. **생활 습관** (3-4문항)
   - 집안일 분담 의식
   - 요리 능력
   - 정리정돈
   - 시간 관리

5. **관계 능력** (3-4문항)
   - 의사소통 스킬
   - 갈등 해결 능력
   - 애정 표현
   - 가족관계

#### 2.1.2 질문 형식
- **객관식**: 4-5개 선택지
- **슬라이더**: 정도를 표현 (0-100)
- **이미지 선택**: 시각적 선호도
- **순위 매기기**: 우선순위 파악

### 2.2 결과 시스템

#### 2.2.1 점수 체계
- **총점**: 100점 만점
- **등급 시스템**:
  - S급 (95-100): "결혼 끝판왕"
  - A급 (85-94): "결혼 준비 완료"
  - B급 (70-84): "매력 충분"
  - C급 (55-69): "조금만 더 노력"
  - D급 (40-54): "자기계발 필요"
  - F급 (0-39): "솔로가 편해요"

#### 2.2.2 결과 페이지
- **메인 점수**: 큰 숫자와 등급 표시
- **레이더 차트**: 5개 카테고리별 점수 시각화
- **장단점 분석**: AI 생성 맞춤 코멘트
- **개선 팁**: 카테고리별 조언
- **재미 요소**: 
  - "당신과 어울리는 연예인"
  - "결혼하면 OO형 배우자"
  - "당신의 결혼 예상 시기"

### 2.3 공유 기능

#### 2.3.1 SNS 공유
- **카카오톡**: 결과 이미지 + 링크
- **인스타그램**: 스토리용 이미지 생성
- **페이스북**: 자동 포스팅
- **트위터(X)**: 간단 결과 + 해시태그

#### 2.3.2 공유 콘텐츠
- **이미지 카드**: 
  - 점수와 등급
  - 짧은 평가 문구
  - QR코드 또는 링크
- **공유 문구 예시**:
  - "나의 결혼매력점수는 85점! A급 결혼준비완료! 당신의 점수는?"
  - "#결혼매력평가 #MAAS #결혼준비"

### 2.4 추가 기능

#### 2.4.1 랭킹 시스템
- **일일 랭킹**: 오늘의 TOP 10
- **주간 랭킹**: 이번 주 최고 점수
- **명예의 전당**: 역대 최고 점수

#### 2.4.2 리테스트
- **재도전 버튼**: 즉시 재시도 가능
- **향상도 표시**: 이전 점수 대비 변화

#### 2.4.3 친구 비교
- **링크 공유**: 친구에게 도전장
- **비교 결과**: 나 vs 친구 점수 비교

## 3. 사용자 경험 (UX)

### 3.1 사용자 플로우
1. **랜딩 페이지** → 매력적인 헤드라인과 시작 버튼
2. **기본 정보 입력** → 닉네임, 성별, 나이대 (선택)
3. **테스트 진행** → 프로그레스바로 진행상황 표시
4. **로딩 화면** → "AI가 분석 중..." (기대감 조성)
5. **결과 페이지** → 점수와 상세 분석
6. **공유/재도전** → SNS 공유 또는 재시도

### 3.2 디자인 원칙
- **모바일 우선**: 세로 모드 최적화
- **원페이지**: 스크롤 기반 진행
- **애니메이션**: 부드러운 전환 효과
- **색상**: 파스텔톤의 친근한 색상
- **타이포그래피**: 읽기 쉬운 폰트

### 3.3 로딩 최적화
- **레이지 로딩**: 필요한 부분만 순차 로딩
- **프리로딩**: 다음 질문 미리 준비
- **최소 대기**: 3초 이내 페이지 로드

## 4. 기술 요구사항

### 4.1 프론트엔드
- **프레임워크**: Next.js 14 (App Router)
- **스타일링**: Tailwind CSS + shadcn/ui
- **상태관리**: Zustand
- **애니메이션**: Framer Motion
- **차트**: Recharts (레이더 차트)

### 4.2 백엔드 (선택적)
- **API**: Next.js API Routes
- **데이터베이스**: Supabase 또는 Firebase
- **인증**: 선택적 (익명 사용 가능)

### 4.3 배포
- **호스팅**: Vercel
- **도메인**: maas.fun (예시)
- **CDN**: Vercel Edge Network

### 4.4 분석
- **Google Analytics 4**: 사용자 행동 분석
- **Hotjar**: 히트맵 분석
- **카카오 픽셀**: 카카오 광고 추적

## 6. 성공 지표 (KPI)

### 6.1 단기 (3개월)
- **DAU**: 1,000명
- **완료율**: 80% 이상
- **공유율**: 30% 이상
- **재방문율**: 20% 이상

### 6.2 중기 (6개월)
- **DAU**: 10,000명
- **MAU**: 100,000명
- **바이럴 계수**: 1.5 이상
- **평균 세션 시간**: 5분 이상

### 6.3 장기 (1년)
- **DAU**: 50,000명
- **MAU**: 500,000명
- **수익화**: 월 1,000만원
- **브랜드 인지도**: 타겟층 30%
